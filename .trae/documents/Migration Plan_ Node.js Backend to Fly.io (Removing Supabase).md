Vou executar o plano de migração para Fly.io em 7 fases estritas:

1. **Limpeza Total do Supabase**: Remover dependências e arquivos.
2. **Schema Prisma Limpo**: Atualizar para modelo de usuário simplificado.
3. **Infra Fly.io**: Inicializar app e banco de dados.
4. **Migration**: Criar banco zerado.
5. **Auth JWT**: Implementar login/registro local.
6. **Deploy**: Publicar e validar.
7. **Validação Final**: Aguardar aprovação.
